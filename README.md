 Инструмент для работы с шифром Цезаря
​В этом репозитории лежит простая программа на Python, которую я написал для шифрования и дешифрования текста. В основе лежит классический шифр Цезаря, где каждая буква заменяется на другую с определенным шагом по алфавиту.

 ​Что умеет программа
​Ты можешь как зашифровать свое сообщение, так и разгадать чужое, если знаешь шаг сдвига.
 ​Программа сохраняет регистр: если ты ввел заглавную букву, на выходе тоже будет заглавная.
 ​Твои знаки препинания, цифры и пробелы останутся на своих местах — алгоритм меняет только буквы.
 ​Я добавил проверку ошибок, так что если ты случайно введешь текст там, где нужно число, программа не вылетит, а попросит тебя повторить ввод.
 ​Тебе не нужно перезапускать скрипт для каждого слова, так как всё работает в удобном цикле.

 ​Как это запустить
​Для работы тебе понадобится Python версии 3.6 или выше.
​Скачай файл со скриптом в любую папку на своем компьютере.
​Открой терминал в этой же папке.
​Введи команду для старта:
python main.py
 
 ​Как пользоваться
Когда ты запустишь программу, она предложит тебе выбрать один из вариантов:
​encrypt: если хочешь зашифровать текст.
​decrypt: если тебе нужно расшифровать сообщение.
​exit: если ты решил закончить работу.
​После выбора режима просто введи свой текст и укажи число (сдвиг). Программа сразу выдаст результат и предложит продолжить работу.
 
 ​Как устроен код
​В основе лежит математическая формула с использованием остатка от деления. Это нужно для того, чтобы алфавит превратился в круг: когда сдвиг доходит до последней буквы, он автоматически перескакивает на первую. Я разделил код на функции, чтобы его было легче читать и дорабатывать.
